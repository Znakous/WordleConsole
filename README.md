# Simple console Worlde game
Implemented in python, compatible with MacOS, others untested
## Gameplay
${\color{green}Green}$
letter means a correct letter on ${\color{green}\underline{correct \ place}}$

${\color{#8b583e}Brown}$
letter means a correct letter on ${\color{#8b583e}\underline{incorrect \ place}}$

## Setup
1) Clone this repository using

```
git clone https://github.com/Znakous/WordleConsole.git 
```

2) Change working directory

```
cd WordleConsole
```

4) Install the needed dependency

```
pip install rich
```
4) Run a game using

```
python main.py
```
⬆️⬆️⬆️ Common run

 ```
 python main.py rus
```
⬆️⬆️⬆️ this will run game in Russian
